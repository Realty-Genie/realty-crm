---
title: Create Workspace
description: Create a new workspace
---

Create a new workspace. The creating user becomes the owner with full permissions.

<EndpointBadge method="POST" path="/api/v1/workspace/create" auth={true} />

## Request Headers

```http
Authorization: Bearer <access_token>
```

## Request Body

```json
{
  "name": "string"
}
```

## Parameters

| Field  | Type   | Required | Description                      |
| ------ | ------ | -------- | -------------------------------- |
| `name` | string | Yes      | Workspace name (min 1 character) |

## Response

Returns `201 Created` on success.

```json
{
  "_id": "507f1f77bcf86cd799439011",
  "name": "My Workspace",
  "type": "SOLO",
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

When a workspace is created, an OWNER membership is automatically created for the creator.

## Workspace Types

| Type   | Description                     |
| ------ | ------------------------------- |
| `SOLO` | Single user workspace (default) |
| `TEAM` | Multi-user workspace            |

<Callout type="info">
  The workspace type is automatically set to `SOLO` when created. It can be
  changed later through the memberships system.
</Callout>
