Realty CRM is a powerful backend API for real estate customer relationship management. It provides a complete authentication system, workspace management, and team collaboration features.

## Features

- **Email & Password Authentication** - Secure registration and login with JWT tokens
- **Google OAuth** - Social login with Google accounts
- **Workspace Management** - Create and manage workspaces for different projects
- **Team Collaboration** - Invite members to workspaces with role-based access
- **Role-Based Access Control** - Admin and user roles with granular permissions
- **Token Management** - Access and refresh token system with automatic renewal

## Tech Stack

| Technology | Purpose            |
| ---------- | ------------------ |
| Bun        | JavaScript runtime |
| Express.js | API framework      |
| MongoDB    | Database           |
| JWT        | Authentication     |
| Zod        | Request validation |
| TypeScript | Type safety        |

## Base URL

```
http://localhost:5000/api/v1
```
